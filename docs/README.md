# SafeTransit Documentation

This directory is the single parent for project documentation.

## Structure

- `docs/project/`: planning and pitch docs
- `docs/frontend/`: frontend design, implementation, and build guides
- `docs/frontend/guidelines/`: UI/UX guidance references

## Key Documents

- `docs/project/execcution_plan.md`
- `docs/project/pitch.md`
- `docs/frontend/PROJECT_OVERVIEW.md`
- `docs/frontend/FRONTEND_STATE.md`
- `docs/frontend/COMPLETE_BUILD_GUIDE.md`

## Related Directories

- Runtime backend code: `backend/`
- Runtime frontend code: `frontend/`
- Data exports and analysis files: `exports/`
